## getJobs
A way to get the list of jobs a player has
```lua showLineNumbers
exporst["br_multiJobs"]:getJobs(target)
```
- target: `integer`
  - The server id of the player that you want the list of jobs

It will return an array in this form
```lua showLineNumbers
{ [1] = "job1", [2] = "job2", [3] = "job3" }
```
