# Installation

<Steps>
### Download a [release](https://github.com/Brodino96/br_itemcreator/releases/latest) or build it yourself
```shell filename="console" showLineNumbers
git clone https://github.com/Brodino96/br_itemcreator
cd br_itemcreator
bun install
bun run prod
```

You can optionally run `./release.ps1` (if on windows) to generate a folder with every file in the right place

### Update the [configs](https://docs.brodino.net/br_itemcreator/config) using the file `config.json`
</Steps>
