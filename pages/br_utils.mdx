# br_utils

This resource is a utility package made for FiveM  
Every features was made by my usemind in focus so it's possible that not everything will be suited for you  
The script is fully configurable and open source so you'll be able to pick and choose what you'll need on your server

Feel free to make any [suggestion](https://github.com/Brodino96/br_utils/issues) or any [modification](https://github.com/Brodino96/br_utils/pulls) you want

## Installation

import { Steps } from "nextra/components"

<Steps>
### Download a [release](https://github.com/Brodino96/br_utils/releases/latest) or clone the repository
```shell filename="console" showLineNumbers
git clone https://github.com/Brodino96/br_utils.git
```

### Configure the script using the file `config.lua`

### Add this line to your `server.cfg` to start the script
```shell filename="server.cfg" showLineNumbers
start br_utils
```

### Enjoy!
</Steps>
