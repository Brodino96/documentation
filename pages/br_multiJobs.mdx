## Installation

import { Steps } from "nextra/components"

<Steps>
### Download a [release](https://github.com/Brodino96/br_multiJobs/releases/latest) or clone the repository
```shell filename="console" showLineNumbers
git clone https://github.com/Brodino96/br_multiJobs
```

### Configure the script using the file `config.lua`

### Add this line to your `server.cfg` to start the script
```shell filename="server.cfg" showLineNumbers
start br_multiJobs
```

### Enjoy!
</Steps>
