Roleplay Simple Voicechat is an addon that expands the functionality of Simple Voice Chat,
with features designed specifically for roleplaay servers. Whether you are running a medieval kingdom,
a modern city, or a post-apocalyptic world, this addon will help with character interactions.

***

## Installation

<Steps>
### Download

**You can simply download a [release](https://github.com/Brodino96/RoleplaySimpleVoiceChat/releases/) or you can build it yourself from scratch**
```sh
git clone https://github.com/Brodino96/RoleplaySimpleVoiceChat
cd RoleplaySimpleVoiceChat
./gradlew build
# ./gradlew.bat if you are on windows
```

### Configuration 

**Update the config file**
- Path: `config/roleplaysimplevoicechat/config.json`

</Steps>
