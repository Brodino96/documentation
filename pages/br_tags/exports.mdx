## Usage
You can use the exports one by one
```lua showLineNumbers
exports["br_tags"]:has(id, tag)
exports["br_tags"]:add(playerId, tag)
exports["br_tags"]:remove(playerId, tag)
```
or  
You can declare a local variable
```lua showLineNumbers
local tags = exports["br_tags"]
tags:has(id, tag)
tags:add(playerId, tag)
tags:remove(playerId, tag)
```
